# 🏧 ATM Management System

A simple ATM Management System built with **Python** that simulates basic banking operations through a command-line interface. The application allows users to log in, check their balance, deposit and withdraw money, transfer funds, and perform basic account management operations.

---

## ✨ Features

- 🔐 User Login Authentication
- 💰 Balance Inquiry
- 💵 Cash Deposit
- 💸 Cash Withdrawal
- 🔄 Fund Transfer
- 👤 User Account Management
- 🖥️ Menu-Driven Command Line Interface

---

## 🛠️ Technologies Used

- Python 3
